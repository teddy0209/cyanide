# Cyanide experimental tweaks

Experimental tweak sources for [Cyanide](https://github.com/zeroxjf/cyanide).
These files are tracked directly in the public Cyanide repository under
`Cyanide/tweaks/experimental/`.

They are open source under AGPL-3.0, matching the main Cyanide project. Beta
entries are free and visible in the app without account linking. In-development
entries can remain visible for continuity, but installation should stay disabled
until someone finishes and verifies them.

See `LICENSE`.
